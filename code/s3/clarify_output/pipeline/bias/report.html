Input data points can be divided into different categories based on their observed and predicted label. For instance, a False Negative (FN) is an input with a positive observed label (fraud = 0) but negative predicted label (fraud != 0). Post-training Bias Metrics¶

Posttraining bias metrics measure imbalances in model predictions across different inputs. The figure below shows how different posttraining metrics target different types of imbalances over inputs. For a detailed description of these types, see Learn How Amazon SageMaker Clarify Helps Detect Bias. Bias can also result form imbalances in the model outcomes even when the facet value is not considered. The metric computing these imbalances is GE.
